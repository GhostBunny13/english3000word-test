## ติดตั้ง

```bash
git clone https://github.com/GhostBunny13/english3000word-test.git
cd english3000word-test
```
```bash
pip install -r requirements.txt
```
##### หรือ
```bash
pip3 install -r requirement.txt
```

```bash
python main.py
```

python3 main.py
